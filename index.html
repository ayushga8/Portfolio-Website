Hi! My name is Ayush. I'm a full-stack web developer and a third-year B.Tech Computer Science student at Poornima University, Jaipur. My journey into web development started in 2024 — and what began as curiosity quickly turned into a serious craft.


I moved past static pages early on. Building backend logic, designing REST APIs, and connecting everything with a database is where I actually enjoy the process. Working on real projects taught me more about architecture and design than any textbook could.


These days my focus is on building production-ready applications. I work across Django, React 19, Express.js, Firebase, and PostgreSQL, and I've shipped projects deployed on Google Cloud Run, Docker, and Vercel. I care about things like test coverage, clean API design, and systems that actually hold up — not just demos that look good in a screenshot.


Outside the code editor I shoot photography, write tech blogs, and follow cricket. I'm always looking to apply my skills to real-world problems, contribute to ambitious teams, and keep growing as a software engineer.
